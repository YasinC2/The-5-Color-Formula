You need 5 colors

- 1 primary color to highlight important elements
(number 5 & button)
- 3 for text, background and second background
(primary-light, primary-dark & base colors)
- And an optional secondary color to add a bit of personality
(the hue slider)

How it works? GitHub

Hue

:root {
    --color-primary: hsl(0, 80%, 20%);
    --color-secondary: hsl(180, 20%, 40%);
    --color-primary-light: hsl(60, 50%, 90%);
    --color-primary-dark: hsl(60, 50%, 10%);
    --color-base: hsl(300, 50%, 95%);
}
.dark {
    --color-primary: hsl(0, 80%, 80%);
